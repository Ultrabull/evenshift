# evenshift

EvenShift — an acuity-balanced nurse assignment board for hospital units. A charge nurse scores
each patient once, taps **Make Assignment**, and gets a fair, geography-aware board in seconds.

- **Live app:** https://ultrabull.github.io/evenshift/
- **Google Sheets edition:** the same engine and rules inside your hospital's Google Workspace —
  easy IT approval, no external services. See [google-sheets/SETUP.md](google-sheets/SETUP.md).
- **Where this is headed:** see [ROADMAP.md](ROADMAP.md) for the 5–10 year vision.
